- 塑料撬棒，避免划伤外壳
- 尖嘴钳（可选），处理卡扣
- 异丙醇和棉签，清洁接触面

## 维修步骤

### 1. 评估上盖状况
1. 检查损坏程度：裂缝、缺口、发黄面积
2. 检查卡扣完整性：是否有断裂的卡扣
3. 检查螺丝孔：是否有滑牙或损坏
4. 决定更换还是修复：小损伤可修复，大损伤建议更换

### 2. 拆卸旧上盖
1. 卸下底部6颗螺丝（注意：2颗在脚垫下面）
2. **重要技巧**：从后部开始分离外壳
3. 用塑料撬棒轻轻撬开卡扣（共8个卡扣）
4. 注意排线：上盖上有电源指示灯排线
5. 断开指示灯排线（轻轻拔下）
6. 完全取下旧上盖

### 3. 准备新上盖
1. **颜色匹配**：选择与原装一致的颜色（灰、黑、蓝等）
2. **版本匹配**：美版、日版、欧版可能有细微差异
3. 清洁新上盖：用异丙醇擦拭内外表面
4. 检查卡扣：确保所有卡扣完整无缺
5. 测试指示灯窗口：确保透光良好

### 4. 处理指示灯排线
1. 从旧上盖上取下指示灯组件（如有需要）
2. 清洁指示灯窗口和LED
3. 测试指示灯：用3V电池测试LED是否亮
4. 安装到新上盖：对准位置，轻轻按压

### 5. 安装新上盖
1. 连接指示灯排线到主板
2. 对准下盖，先对齐后部卡扣
3. 轻轻按压，让卡扣逐个扣合
4. **注意顺序**：从后向前，从左到右
5. 检查所有卡扣是否完全扣合
6. 上紧底部6颗螺丝

### 6. 最终检查
1. 检查外观：缝隙是否均匀，有无翘起
2. 测试指示灯：开机时红色灯是否亮
3. 检查稳定性：按压各部位有无异响
4. 功能测试：运行游戏，确保无干扰
