# Wii 视频输出维修完全指南

![Wii](images/wii.webp)

> 你的Wii开机有声音但没图像，或者画面闪烁、颜色异常、分辨率不对？最崩溃的是玩《塞尔达传说：黄昏公主》时突然黑屏！Wii的视频输出系统支持多种格式，从480i到480p，还有特殊的480p逐行扫描。别担心，大多数视频问题都能找到原因并修复。

## 所需工具
- 十字螺丝刀，PH0规格
- 三角螺丝刀（Y字头），Wii专用
- 万用表，带频率测试功能
- 示波器（强烈推荐）
- 电烙铁和焊锡
- 热风枪（芯片维修需要）
- 电子清洁剂
- 棉签和异丙醇
- 各种视频线测试（AV、分量、S端子）
- 防静电手套和工作垫

## 维修步骤

### 1. 安全拆机，了解视频系统架构
1. 拔掉所有线材，记住传感器条位置
2. 卸下底部所有螺丝（注意脚垫下的隐藏螺丝）
3. 打开外壳，小心WiFi和蓝牙天线
4. 找到视频相关部分：
   - AV接口（主板后部）
   - 视频编码芯片
   - GPU（Hollywood芯片）
   - 视频输出相关元件

### 2. Wii视频系统详解
#### 视频输出类型：
1. **复合视频**：标准AV接口，480i隔行扫描
2. **S端子**：分离亮度和色度，画质较好
3. **分量视频**：YPbPr，支持480p逐行扫描
4. **RGB**：需要改装，欧洲版支持

#### 关键芯片：
1. **GPU**：Hollywood芯片，集成视频处理
2. **视频编码器**：将数字信号转为模拟信号
3. **输出驱动**：增强信号驱动能力
4. **模式切换**：控制输出格式切换

### 3. 系统诊断流程
#### 第一步：基础检查
1. **线缆测试**：用已知好的视频线测试
2. **电视测试**：换电视或输入接口测试
3. **分辨率测试**：尝试480i和480p（如果支持）
4. **接口清洁**：清洁所有视频接口

#### 第二步：信号追踪
用万用表和示波器追踪信号：

**AV接口针脚定义**：
- 针脚1：复合视频信号
- 针脚2：右声道音频
- 针脚3：左声道音频
- 针脚4：地线
- 针脚5：S端子亮度（如果支持）
- 针脚6：S端子色度（如果支持）

**分量接口**：独立的Y、Pb、Pr接口

#### 第三步：关键点测试
1. **接口电压**：测量接口相关电压
2. **信号波形**：用示波器看视频信号
3. **时钟信号**：检查视频时钟是否正常
4. **同步信号**：检查行场同步信号

### 4. 常见故障维修
#### 故障1：完全无图像（有声音）
1. 检查视频线是否完好
2. 测试复合视频输出电容
3. 检查视频编码芯片供电
4. 测量GPU视频输出信号
5. 检查模式切换电路

#### 故障2：图像闪烁/不稳定
1. 检查视频信号耦合电容
2. 检查同步信号相关电路
3. 检查电源滤波是否良好
4. 检查是否有干扰源
5. 检查视频时钟稳定性

#### 故障3：颜色异常
1. 检查色度信号相关电路
2. 检查视频编码芯片
3. 检查分量输出的Pb/Pr通道
4. 检查电视制式设置（NTSC/PAL）
5. 检查色度滤波电路

#### 故障4：只有480i没有480p
1. 检查分量输出电路
2. 检查模式切换芯片
3. 检查软件设置（需要进入设置开启480p）
4. 检查游戏本身是否支持480p
5. 检查电视是否支持480p输入

#### 故障5：画面有干扰条纹
1. 检查电源滤波电容
2. 检查视频信号屏蔽
3. 检查接地是否良好
4. 检查附近是否有干扰源
5. 检查视频编码芯片供电

### 5. 视频编码芯片维修
如果确定是编码芯片问题：

#### 芯片识别：
Wii常用视频编码芯片：
- **CXA2075Q**：索尼制，常见
- **其他型号**：不同批次可能不同

#### 维修步骤：
1. **供电测试**：测量所有供电引脚
2. **时钟测试**：检查时钟输入是否正常
3. **信号测试**：测试输入输出信号
4. **芯片更换**：需要BGA焊接技术
5. **固件检查**：检查视频相关固件设置

### 6. 分量输出特别维修
Wii的分量输出需要特别注意：

1. **分量接口检查**：检查三个RCA接口
2. **分量信号测试**：分别测试Y、Pb、Pr信号
3. **低通滤波检查**：检查分量输出的滤波电路
4. **驱动电路检查**：检查分量信号驱动电路
5. **模式切换检查**：检查480i/480p切换电路

### 7. 软件相关问题
有时是软件而非硬件问题：

1. **分辨率设置**：需要在Wii设置中开启480p
2. **电视制式**：NTSC和PAL制式问题
3. **游戏兼容性**：某些游戏只支持480i
4. **系统版本**：更新系统可能解决视频问题
5. **区域设置**：不同区域视频标准不同

## Wii视频技术细节

### 🎮 视频规格
- **分辨率**：480i/480p（NTSC），576i（PAL）
- **色彩空间**：YPbPr（分量），Y/C（S端子），Composite（复合）
- **输出格式**：支持逐行和隔行扫描
- **抗锯齿**：硬件支持2x多重采样抗锯齿

### 📺 不同输出方式画质对比
1. **复合视频**：最差，有色彩串扰，480i
2. **S端子**：较好，亮色分离，480i
3. **分量视频**：很好，色彩准确，支持480p
4. **RGB**：最佳，需要改装

### 🔧 型号与区域差异
- **NTSC-U**：美版，480i/480p
- **NTSC-J**：日版，480i/480p
- **PAL**：欧版/澳版，576i/480p（部分）
- **视频编码**：不同区域可能使用不同芯片

## 维修技巧与注意事项

### ⚠️ 安全注意事项
1. **静电防护**：视频芯片对静电敏感
2. **焊接技巧**：BGA芯片需要专业设备
3. **测试顺序**：先低压测试，逐步增加
4. **示波器使用**：注意接地，避免短路
5. **高压部分**：注意电源相关电路安全

### 🔧 专业技巧
1. **信号注入法**：从后往前注入信号测试
2. **对比法**：与正常主板对比测量
3. **加热法**：对可疑芯片局部加热测试
4. **代换法**：用已知好元件代换测试
5. **软件诊断**：进入工程模式测试

5. **优点**：即插即用，画质优秀
6. **缺点**：价格较高，需要额外设备
