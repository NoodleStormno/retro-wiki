## Safety Precautions

Read these before any repair:

1. Power off: Always disconnect power before any repair
2. Anti-static: Wear anti-static gloves when handling delicate circuits
3. Tool preparation: Use appropriate tools to avoid damaging screws
4. Part storage: Keep all screws and parts organized during disassembly
5. Ribbon cable care: Handle ribbon cables gently to avoid breaking
