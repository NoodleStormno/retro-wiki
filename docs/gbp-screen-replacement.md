- **热风枪或电吹风**：软化旧胶（可选）
- **万用表**：测量排线连接
- **放大镜台灯**：看清排线连接
- **防静电手套**：保护液晶屏

## 详细维修步骤

我在秋叶原修了20年GameBoy，总结出这套方法。跟着做，别着急。

### 第一步：安全拆机

拆机最怕的就是损坏屏幕排线。慢慢来：

1. 先卸下电池盖，取出里面的2节AAA电池
2. 拆除6颗螺丝，4颗Y字型螺丝在背面，还有2颗十字螺丝藏在电池仓的贴纸下面
3. 用塑料撬棒从充电口那里轻轻撬开外壳
4. **特别小心**：断开主板和液晶屏连接的排线
   - 用指甲或者塑料片轻轻撬起排线锁扣
   - 不要用金属工具，会划伤排线
   - 不要用力拉扯，排线很脆弱
5. 卸下固定主板的2颗螺丝，取出主板

### 第二步：旧屏幕拆除

取出主板后，开始拆除旧屏幕：

1. **检查屏幕固定方式**：
   - 原装屏幕通常用双面胶固定
   - 可能有塑料卡扣
   - 注意观察再动手

2. **软化胶水**（如果需要）：
   - 用热风枪低温（80-100度）吹屏幕边缘
   - 或者用电吹风热风档
   - 让胶水软化

3. **拆除旧屏幕**：
   - 用塑料撬棒从边缘轻轻撬起
   - 如果阻力很大，继续加热
   - 不要用蛮力，会损坏屏幕框架

4. **清理框架**：
   - 用异丙醇清理框架上的旧胶
   - 确保框架干净平整
   - 检查框架是否有损坏

### 第三步：新屏幕准备

新屏幕安装前要做好准备：

1. **检查新屏幕**：
   - 检查屏幕是否有损坏
   - 检查排线是否完好
   - 测试屏幕（如果可能）

2. **屏幕型号确认**：
   - Game Boy Pocket原装屏幕型号
   - 夏普LM24014T驱动
   - 尺寸：2.6英寸
   - 分辨率：160x144

3. **兼容性测试**：
   - 如果不确定是否兼容
   - 先连接测试显示
   - 确认正常再安装

### 第四步：新屏幕安装

这是最关键的一步：

1. **准备双面胶**：
   - 剪裁合适尺寸的双面胶
   - 贴在屏幕框架上
   - 注意留出排线位置

2. **屏幕对位**：
   - 将屏幕放入框架
   - 对准所有边缘
   - 确保完全平整

3. **固定屏幕**：
   - 轻轻按压屏幕四周
   - 确保双面胶完全粘合
   - 检查是否有气泡

4. **排线连接**：
   - 将屏幕排线连接到主板
   - 对准接口，轻轻插入
   - 锁紧排线锁扣

### 第五步：初步测试

安装后立即测试：

1. **连接电池测试**：
   - 先不要装机，直接连接电池
   - 开机测试屏幕显示
   - 检查是否有坏点、线条

2. **显示测试**：
   - 测试全白画面
   - 测试全黑画面
   - 测试游戏画面

3. **功能测试**：
   - 测试对比度调节
   - 测试屏幕在不同角度下的可视性

### 第六步：背光安装（可选）

如果你要安装背光：

1. **背光类型选择**：
   - 前光：光线从前面照射
   - 背光：光线从后面照射
   - 侧光：光线从侧面导入

2. **安装步骤**：
   - 拆除原装反光板
   - 安装导光板
   - 安装LED灯条
   - 连接电源

3. **注意事项**：
   - 背光会增加耗电
   - 可能影响屏幕对比度
   - 安装难度较高

### 第七步：最终组装与测试

1. 屏幕测试通过后，开始最终组装
2. 安装主板，固定螺丝
3. 合上外壳，先不上螺丝测试
4. **最终测试项目**：
   - 开机显示测试
   - 游戏运行测试
   - 对比度调节测试
   - 可视角度测试
   - 长时间运行测试（1小时）
5. 测试通过后，上紧所有螺丝
6. 贴回电池仓贴纸
7. 贴上屏幕保护膜

## 老师傅的叮嘱

在秋叶原开店20年，客人常问这些问题，我一个个回答你。

### 为什么Game Boy Pocket屏幕容易损坏？

三个主要原因。第一是物理损伤，摔落或挤压导致屏幕碎裂。第二是液晶老化，20多年的液晶可能漏液或失效。第三是环境因素，高温或潮湿影响屏幕。

### 原装屏幕和兼容屏幕有什么区别？

区别很大。原装屏幕：
- 显示效果最好，对比度高
- 可视角度好
- 质量稳定
- 但价格贵，难找

兼容屏幕：
- 价格便宜
- 容易购买
- 但显示效果可能差一些
- 可能有坏点或线条

### 值不值得换屏幕？还是直接换机器？

这得算经济账。换屏幕成本2,500-4,000日元。买一台成色好的二手Pocket要5,000-8,000日元。但如果你的机器有特殊意义，或者成色很好，换屏幕更值得。

### 如何保护新屏幕？

我建议立即贴上保护膜。使用时避免尖锐物体接触屏幕。存放时屏幕朝上，不要压到东西。定期清洁屏幕，用专用清洁布。

## 维修难度

技术难度四颗星，需要细心操作。工具要求三颗星，基本工具就行。时间成本大概2-3小时，第一次尝试可能更久。成功率很高，90%以上。

## 我的终极建议

在秋叶原修了20年，我总结出几条经验：

1. 先测试后安装，确认屏幕正常再装
2. 排线要小心，这是最脆弱的部分
3. 清洁要彻底，灰尘会影响显示
4. 对位要准确，歪了很难调整
5. 测试要全面，所有功能都要测试
