# NDS转轴断裂更换外壳图文

> **故障现象**：NDS上盖转轴断裂，屏幕无法固定角度，晃动严重，有时排线因过度弯折出现显示问题。

## 所需工具
- Y字螺丝刀（特殊规格）
- 十字螺丝刀
- 吸盘
- 撬棒套装
- 新外壳（带转轴）
- 电烙铁（如需焊接排线）

## 维修步骤

### 1. 拆卸旧外壳
1. 卸下背部所有Y字螺丝
2. 用吸盘分离触摸屏
3. 小心断开所有排线连接
4. 取出主板和屏幕组件

### 2. 更换转轴部件
1. 将主板安装到新外壳中
2. 对准转轴安装位置
3. 固定转轴螺丝（不要过紧）
4. 测试转轴开合顺畅度

### 3. 安装屏幕组件
1. 将上下屏幕安装到新外壳
2. 连接所有排线（注意方向）
3. 整理排线走向，避免挤压
4. 固定屏幕边框

### 4. 最终测试
1. 暂时不装螺丝，测试开机
2. 检查屏幕显示和触摸功能
3. 测试转轴开合和固定角度
4. 确认无误后安装所有螺丝

## 注意事项
- NDS排线非常脆弱，操作要轻柔
- 转轴螺丝力度要适中，过紧会卡顿
- 安装前测试所有按钮和接口
- 建议同时更换磨损的按键胶垫

## 常见问题
1. **排线损坏**：更换排线需焊接技巧
2. **转轴不顺畅**：适当调整螺丝松紧
3. **屏幕亮点**：安装时压力不均导致
4. **按键失灵**：清洁或更换按键胶垫
