# PSP Go Bottom Plate Replacement Complete Guide

## Overview

The PSP Go bottom plate protects important ports and internal components including charging port, headphone jack, and memory card slot. This guide provides detailed instructions for replacing the PSP Go bottom plate.

## Required Tools

- Cross-head screwdriver (PH00)
- Plastic spudger
- New PSP Go bottom plate
- Anti-static wrist strap (recommended)
- Tweezers (optional)

## Safety Precautions

1. **Power off completely**: Ensure PSP Go is fully powered down
2. **Port protection**: Important ports under bottom plate, handle carefully
3. **Clip care**: Bottom plate has multiple precision clips, avoid excessive force
4. **Dust/static protection**: Operate in clean environment

## Detailed Steps

### Step 1: Preparation

1. Completely power off PSP Go
2. Remove all cables (charger, headphones, etc.)
3. Remove Memory Stick Micro card
4. Clean work area to avoid dust ingress

### Step 2: Remove Back Cover

1. Place PSP Go face down
2. Remove 4 screws from back cover
3. Use plastic spudger to gently pry open back cover
4. Carefully remove back cover, place in safe location

### Step 3: Disconnect Battery

1. Locate battery connector
2. Use plastic spudger to gently disconnect battery
3. Ensure device is completely powered off

### Step 4: Remove Motherboard

1. Remove 6 screws securing motherboard
   - Note: Screw lengths may differ, record positions
2. Gently lift motherboard, but don't completely remove
3. Avoid pulling any ribbon cables

### Step 5: Remove Old Bottom Plate

1. Bottom plate secured by screws and clips
2. Remove 3 screws securing bottom plate
3. Use plastic spudger to release clips starting from edges
4. Carefully remove old bottom plate
5. Check ports under bottom plate for dust or damage

### Step 6: Install New Bottom Plate

1. Clean bottom plate installation area
2. Align new bottom plate with body
3. Starting from one side, gently press clips
4. Ensure all clips properly engaged
5. Tighten 3 securing screws

### Step 7: Reinstall Motherboard

1. Return motherboard to original position
2. Tighten 6 screws in original order
3. Avoid overtightening

### Step 8: Reconnect Battery

1. Reconnect battery cable
2. Ensure secure connection

### Step 9: Install Back Cover

1. Place back cover, align all clips
2. Replace 4 screws
3. Check gap between bottom plate and back cover is even

## Post-Installation Checks

### Visual Inspection
1. Check bottom plate installed flat
2. Check all port openings aligned
3. Check all screws tightened

### Function Testing
1. Power on to test device functions
2. Test charging port works normally
3. Test headphone jack audio output
4. Test memory card slot read/write
5. Check sliding mechanism smoothness

## Port Functions Detailed

### Charging Port (DC IN)
- **Location**: Right side of bottom plate
- **Function**: 5V DC input
- **Check**: Insert charger to test charging indicator

### Headphone Jack
- **Location**: Left side of bottom plate
- **Function**: 3.5mm audio output
- **Check**: Insert headphones to test audio output

### Memory Card Slot
- **Location**: Center of bottom plate
- **Function**: Memory Stick Micro storage
- **Check**: Insert memory card to test read/write

## Common Issues and Solutions

### Issue 1: Port misalignment
- **Possible cause**: Bottom plate installation offset
- **Solution**: Reinstall, ensure perfect alignment

### Issue 2: Clips won't engage
- **Possible cause**: Clips damaged or wrong installation angle
- **Solution**: Check clip condition, adjust installation angle

### Issue 3: Screw holes don't match
- **Possible cause**: Bottom plate model mismatch
- **Solution**: Confirm purchased PSP Go-specific bottom plate

### Issue 4: Function abnormalities
- **Possible cause**: Internal components damaged during installation
- **Solution**: Recheck internal connections, professional repair if damaged

## Maintenance Recommendations

2. **Headphone jack**: Regular cleaning, avoid foreign objects
3. **Memory card slot**: Carefully insert/remove cards, avoid pin damage
