# PSP Go Game Buttons Replacement Complete Guide

## Overview

PSP Go game buttons include the four main action buttons (△○×□), which are core components for game control. After prolonged use, they may develop issues like unresponsiveness, weak rebound, or sticking. This guide provides detailed instructions for replacing PSP Go game buttons.

## Required Tools

- Phillips screwdriver (PH00)
- Plastic spudger
- Tweezers
- New PSP Go game button set (△○×□)
- Anti-static wrist strap (recommended)

## Safety Precautions

1. **Power off**: Ensure PSP Go is completely powered down
2. **Static protection**: Discharge static electricity before handling
3. **Careful with flex cables**: Button circuit board flex cables are fragile
4. **Silicone pad protection**: Handle rebound-providing silicone pads carefully

## Button Structure Analysis

### Physical Structure
1. **Button caps**: Parts users directly press (△○×□)
2. **Silicone pads**: Provide button rebound force
3. **Circuit board**: Detects button presses
4. **Mounting frame**: Secures button positions

### Button Functions
1. **△ button**: Triangle, often used for special actions
2. **○ button**: Circle, often used for confirm/attack
3. **× button**: Cross, often used for cancel/jump
4. **□ button**: Square, often used for defense/functions

## Step-by-Step Instructions

### Step 1: Remove Back Cover

1. Place PSP Go face down
2. Remove 4 screws on back cover
3. Gently pry open back cover with plastic spudger
4. Carefully remove back cover

### Step 2: Disconnect Battery

1. Locate battery connector
2. Disconnect battery using plastic spudger
3. Ensure device is completely powered down

### Step 3: Remove Motherboard

1. Remove 6 screws securing motherboard
2. Note screw positions (lengths may differ)
3. Gently lift motherboard
4. Be careful not to pull on flex cables

### Step 4: Remove Front Panel

1. Remove screws securing front panel
2. Release front panel clips with plastic spudger
3. Carefully remove front panel
4. Expose button area

### Step 5: Remove Old Buttons

1. Locate button circuit board flex cable connector
2. Gently lift flex cable latch with tweezers
3. Remove button flex cable
4. Remove screws securing button frame
5. Remove old button frame
6. Remove old △○×□ buttons

### Step 6: Clean Installation Area

1. Clean button installation area with cotton swabs
2. Check silicone pads for integrity
3. Clean circuit board contacts
4. Ensure area is clean and dust-free

### Step 7: Install New Buttons

1. Install new silicone pads (if included)
2. Place new △○×□ buttons in corresponding positions
3. Note button orientation: △ top, ○ right, × bottom, □ left
4. Install button frame
5. Tighten mounting screws

### Step 8: Connect Flex Cable

1. Connect button circuit board flex cable
2. Ensure cable fully inserted
3. Press latch to secure

### Step 9: Test Buttons

1. Temporarily connect battery
2. Power on and test each button function
3. Test button feel and rebound
4. Check for sticking

### Step 10: Reassemble

1. Install front panel
2. Install motherboard
3. Connect battery
4. Install back cover
5. Tighten all screws

## Common Issues & Solutions

### Issue 1: Button Unresponsive
- **Possible cause**: Silicone pad damaged or circuit board failure
- **Solution**: Replace silicone pad, check circuit board

### Issue 2: Inconsistent Button Feel
- **Possible cause**: Silicone pad improperly installed or thickness varies
- **Solution**: Reinstall silicone pad, ensure evenness

### Issue 3: Button Sticking
- **Possible cause**: Wrong button cap size or too tight installation
- **Solution**: Check button cap dimensions, adjust installation

### Issue 4: False Triggering
- **Possible cause**: Silicone pad too thin or circuit sensitivity issue
- **Solution**: Replace with proper thickness silicone pad
