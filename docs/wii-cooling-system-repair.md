# Wii 散热系统维修完全指南

![Wii](images/wii.webp)

> 你的Wii运行一段时间后异常发热，风扇噪音变大，甚至自动关机保护？或者根本听不到风扇声音，主机烫手！Wii的散热设计比GameCube更先进，但硅脂干裂、风扇老化、灰尘堵塞依然常见。别担心，散热维护相对简单，成功率很高。

## 所需工具
- 十字螺丝刀，PH0规格
- 三角螺丝刀（Y字头），Wii专用
- 精密镊子
- 高质量散热硅脂（信越7921或类似）
- 压缩空气罐或吹气球
- 软毛刷
- 异丙醇和无尘布
- 万用表（测试风扇）
- 润滑油（风扇轴承用）
- 防静电手套

## 维修步骤

### 1. 拆机准备，找到散热系统
1. 拔掉所有线材，记住传感器条位置
2. 卸下底部所有螺丝（注意脚垫下的隐藏螺丝）
3. 打开外壳，小心WiFi和蓝牙天线
4. 找到散热系统——位于主板中央的金属散热片和风扇
5. 注意：需要先拆除光驱和部分屏蔽罩

### 2. 散热系统组件检查
#### 散热片检查：
1. **灰尘堆积**：散热片鳍片间是否被灰尘堵塞
2. **氧化情况**：铜质散热片是否严重氧化
3. **热管检查**：检查热管是否变形、泄漏
4. **固定状态**：散热片是否松动，螺丝是否紧固

#### 风扇检查：
1. **转动测试**：手动拨动风扇叶片，是否顺畅
2. **噪音检查**：转动时是否有异响、摩擦声
3. **轴承检查**：检查轴承是否磨损、缺油
4. **线缆检查**：风扇电源线是否完好，接口是否氧化

#### 硅脂状态检查：
1. **干裂情况**：原厂硅脂是否已干裂成粉末
2. **覆盖情况**：是否还能有效覆盖芯片表面
3. **污染情况**：是否混入灰尘、杂质
4. **厚度检查**：硅脂层是否过厚或过薄

### 3. 深度清洁流程
#### 散热片清洁：
1. 用软毛刷轻轻刷去表面灰尘
2. 用压缩空气从内向外吹散热片缝隙
3. 顽固污渍用异丙醇棉签擦拭
4. 检查每个鳍片是否通畅，无堵塞
5. 清洁热管与散热片连接处

#### 风扇清洁：
1. 用刷子清洁风扇叶片两面
2. 用棉签清洁风扇框架内部
3. 检查并清洁风扇电源接口
4. 给风扇轴承加一滴精密润滑油（非必需）
5. 检查风扇固定螺丝是否松动

#### 主板清洁：
1. 用压缩空气吹去主板灰尘
2. 重点清洁芯片周围和电源部分
3. 检查其他散热相关元件
4. 清洁散热风道，确保通畅

### 4. 硅脂更换指南
这是最重要的步骤，直接影响散热效果：

1. **拆除散热片**：
   - 卸下固定散热片的螺丝（通常4-6颗）
   - 轻轻左右扭动散热片，不要硬拔
   - 小心提起散热片，避免损坏芯片或热管
   - 注意热管连接，不要过度弯曲

2. **清理旧硅脂**：
   - 用无尘布蘸异丙醇擦拭GPU芯片表面
   - 同样清洁散热片底部接触面
   - 确保完全干净，无残留硅脂
   - 检查芯片表面有无划痕、损伤

3. **涂抹新硅脂**：
   - 挤出豌豆大小的硅脂在芯片中央
   - 用刮片或手指套均匀涂抹
   - 厚度约0.3-0.5mm，覆盖整个芯片
   - 不要涂太厚，影响导热效果
   - 确保散热片能均匀压平硅脂

4. **安装散热片**：
   - 对准位置，轻轻放下散热片
   - 对角线顺序拧紧螺丝
   - 力度适中，不要过紧或过松
   - 确保散热片与芯片完全接触

### 5. 风扇维修与更换
#### 风扇测试：
1. 用万用表测量风扇电阻，正常约50-100Ω
2. 直接给风扇接5V电源测试转动
3. 测量电流，正常约0.1-0.15A
4. 测试不同电压下的转速变化

#### 风扇维修：
1. **轴承润滑**：加一滴精密仪器润滑油
2. **叶片清洁**：彻底清洁每个叶片
3. **线缆修复**：如有断裂，焊接修复
4. **框架调整**：如有变形，小心调整

#### 风扇更换：
如果风扇损坏严重：
1. 购买同规格风扇（5V 0.15A 40mm）
2. 注意接口方向和引脚定义
3. 固定风扇，确保与散热片对齐
4. 测试风扇转动方向和风量

### 6. 散热系统测试
1. **空载测试**：开机观察风扇是否转动
2. **温度测试**：运行30分钟，手摸散热片温度
3. **负载测试**：运行《超级马里奥银河》1小时
4. **噪音测试**：在不同负载下听风扇声音
5. **自动关机测试**：连续运行3-4小时
6. **温控测试**：观察风扇是否随温度变化转速

### 7. 风道优化
Wii散热风道优化建议：
1. **进风口清洁**：清洁底部进风口
2. **出风口清洁**：清洁后部出风口
3. **内部风道**：确保内部无遮挡
4. **外部环境**：确保主机周围通风良好

## Wii散热技术细节

### 🎮 散热设计特点
- **主动散热**：温控风扇+铜质散热片
- **热管技术**：使用热管增强导热
- **风道设计**：底部进风，后部出风
- **温控系统**：根据温度调节风扇转速
- **低功耗设计**：整体发热量较小

### 🌡️ 正常温度范围
- **待机温度**：GPU表面35-45°C
- **游戏温度**：GPU表面55-65°C
- **散热片温度**：比GPU低10-15°C
- **安全上限**：GPU 75°C（自动保护）
- **风扇启动**：通常45-50°C开始转动

### 🔧 不同型号差异
- **RVL-001**：散热系统较完善，热管质量好
- **RVL-101**：简化散热，但基本够用
- **Wii Mini**：被动散热，无风扇
- **散热改进**：后期型号散热有所优化

## 维修技巧与注意事项

### ⚠️ 安全注意事项
1. **静电防护**：GPU芯片对静电敏感
2. **硅脂绝缘**：确保硅脂不导电
3. **螺丝力度**：不要拧太紧，避免压坏芯片
4. **热管保护**：不要过度弯曲热管
5. **测试顺序**：先测试再完全组装

### 🔧 专业技巧
1. **硅脂涂抹**：九点法或十字法都可以
2. **清洁技巧**：用牙签清理散热片缝隙
3. **风扇润滑**：一滴足够，不要多
4. **温度监控**：可用红外测温枪或热敏电阻
5. **风量测试**：用纸条测试风扇风量
