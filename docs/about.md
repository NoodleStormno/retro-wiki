# 关于我们

欢迎来到 **Retro Console Wiki**。

我们是一群充满热情的中古游戏爱好者和硬件技术专家，深受日本秋叶原DIY和维修文化的影响。多年来，我们花费了无数时间在Hard Off等商店寻找"Junk"（损坏/废弃）游戏机，并用我们的双手让它们重获新生。

这个Wiki的目的很简单：**防止中古游戏机维修艺术逐渐消失。**

随着现代设备变得越来越难以维修，我们希望为任天堂和索尼等经典游戏机提供最硬核、最准确、最实用的维修指南。从更换GBA的IPS屏幕到诊断Switch的主板短路，我们分享真实的参数、元件成本和逐步拆解教程。

拯救你的游戏机。让像素继续闪烁。

---

**我们的使命**：
- 保存和传承经典游戏机维修知识
- 提供准确、实用的维修指南
- 建立中古游戏爱好者社区
- 推广可维修性和可持续发展理念

**我们的价值观**：
- **准确性**：所有指南都基于实际维修经验
- **实用性**：提供真实可用的解决方案
- **开放性**：知识应该被自由分享
- **热情**：对中古游戏的真挚热爱

**团队背景**：
我们的团队成员来自不同的技术背景，但都拥有共同的热爱：
- 电子工程师和硬件维修专家
- 软件开发和逆向工程爱好者
- 游戏收藏家和历史研究者
- DIY文化和开源硬件倡导者

**项目起源**：
这个项目起源于我们在日本秋叶原的维修经历。在那里，我们看到了大量被丢弃但仍有修复价值的经典游戏机。我们相信，每一台游戏机都值得被修复，每一段游戏历史都值得被保存。

通过这个Wiki，我们希望将这种维修文化传播给更多的人，让更多的经典游戏机能够继续运行，让更多的游戏记忆能够被重温。
