# PSP 1000 屏幕更换完全指南

> 你的 PSP 1000 屏幕出现了坏点、亮线、背光不均匀，或者干脆不显示了。作为 PSP 初代机型，它的 4.3 英寸 TFT 屏幕虽然经典，但经过多年使用难免出现问题。别担心，屏幕更换是 PSP 维修中最常见的项目之一！

## 所需工具
- 十字螺丝刀，PH00规格
- 塑料撬棒或吉他拨片
- 新的 PSP 1000 屏幕总成
- 无水酒精和棉签
- 精密镊子
- 防静电手套和工作垫
- 吹风机或热风枪（可选，用于分离胶条）

## 维修步骤

### 1. 安全拆机准备
1. **完全断电**：取出电池和 UMD 光盘
2. **卸下所有螺丝**：
   - 背面6颗螺丝（4颗在电池仓，2颗在 UMD 仓）
   - 注意螺丝长度不同，做好标记
3. **分离外壳**：使用塑料撬棒从边缘轻轻撬开
4. **小心排线**：打开时注意屏幕排线和按键排线

### 2. 屏幕总成拆卸
1. **找到屏幕排线**：位于主板右侧，白色或棕色排线
2. **打开排线锁扣**：用镊子轻轻抬起锁扣
3. **拔出屏幕排线**：轻轻拉出排线，不要用力过猛
4. **卸下屏幕固定件**：
   - 屏幕四周有金属框架固定
   - 卸下固定螺丝（通常4颗）
5. **取出旧屏幕**：小心取出整个屏幕总成

### 3. 新屏幕安装
1. **检查新屏幕**：确认是 PSP 1000 专用屏幕
2. **清洁安装面**：用无水酒精清洁屏幕安装区域
3. **安装屏幕**：将新屏幕放入框架，对准螺丝孔
4. **固定屏幕**：安装固定螺丝，注意力度均匀
5. **连接排线**：插入屏幕排线，按下锁扣固定

### 4. 屏幕测试
1. **初步测试**：先不装外壳，连接电池开机测试
2. **检查显示**：观察屏幕是否有显示异常
3. **测试触摸**：PSP 1000 无触摸屏，跳过此步
4. **检查背光**：观察背光是否均匀，有无暗角
5. **测试色彩**：显示纯色图片检查坏点和色偏

### 5. 组装与最终测试
1. **清洁内部**：用棉签清理内部灰尘
2. **重新组装**：按照拆机逆序组装外壳
3. **螺丝归位**：注意螺丝长度和位置
4. **最终测试**：完整组装后测试所有功能

## 常见屏幕问题与解决

### 问题1：屏幕有坏点
1. **坏点类型**：
   - 亮点：始终亮着的像素
   - 暗点：始终不亮的像素
   - 色点：显示固定颜色的像素
2. **修复尝试**：使用坏点修复软件（成功率低）
3. **最终方案**：更换屏幕

### 问题2：屏幕亮线
1. **水平亮线**：通常是排线问题
2. **垂直亮线**：可能是屏幕本身损坏
3. **多条亮线**：屏幕驱动芯片故障
4. **维修方案**：更换屏幕总成

### 问题3：背光不均匀
1. **边缘暗角**：背光灯老化或安装问题
2. **局部暗区**：背光导光板问题
3. **整体偏暗**：背光灯寿命到期
4. **解决方案**：更换屏幕或单独更换背光

### 问题4：屏幕闪烁
1. **频率性闪烁**：背光驱动电路问题
2. **随机闪烁**：排线接触不良
3. **颜色闪烁**：屏幕驱动芯片故障
4. **排查步骤**：检查排线→检查主板→更换屏幕

### 🛠️ 屏幕排线处理技巧
屏幕排线非常脆弱，需要特别小心：

#### 排线插拔技巧
1. **完全打开锁扣**：确保锁扣完全抬起
2. **平行拔出**：保持排线平行拔出，不要歪斜
3. **检查金手指**：检查排线金手指是否清洁完好
4. **重新插入**：对准接口，轻轻推到底部

#### 排线保护措施
1. **避免折叠**：不要过度弯曲排线
2. **防静电**：操作前触摸金属物体放电
3. **清洁接口**：用无水酒精清洁排线接口
4. **固定位置**：组装时确保排线位置正确

### 🔧 屏幕框架与胶条处理
PSP 1000 屏幕使用双面胶固定：

#### 旧胶条去除
1. **加热软化**：用吹风机低温加热胶条
2. **慢慢分离**：使用塑料撬棒慢慢分离
3. **清理残胶**：用无水酒精清理残留胶水
4. **避免划伤**：不要使用金属工具

#### 新胶条安装
1. **选择胶条**：使用3M双面胶或专用屏幕胶
2. **裁剪尺寸**：根据屏幕边框裁剪胶条
3. **均匀粘贴**：确保胶条粘贴平整无气泡
4. **按压固定**：安装后按压屏幕确保粘合

### 🎨 屏幕色彩校准
更换屏幕后可能需要进行色彩校准：

#### 软件校准
1. **进入恢复模式**：关机后按 R 键开机
2. **调节色彩**：在 Advanced 中找到色彩调节选项
3. **参考标准**：使用标准色卡图片作为参考
4. **保存设置**：调节后保存设置

#### 硬件限制
1. **色温差异**：不同批次屏幕色温可能不同
2. **伽马值**：兼容屏伽马值可能与原装有差异
3. **可视角度**：兼容屏可视角度可能较小
4. **接受差异**：轻微色差属于正常现象

2. **坏点检测**：仔细观察有无坏点

#### 高级检测
1. **色彩过渡**：显示色彩渐变图片
2. **对比度测试**：显示黑白对比图片
3. **视角测试**：从不同角度观察屏幕
4. **触摸测试**：PSP 1000 无触摸，跳过

### 💡 屏幕保养与维护
延长新屏幕寿命的方法：

#### 日常使用
1. **避免挤压**：不要给屏幕施加压力
2. **清洁方法**：使用专用屏幕清洁布
3. **避免高温**：不要长时间暴露在高温环境
4. **使用保护膜**：贴上高质量屏幕保护膜

#### 长期存放
1. **适当温度**：存放在阴凉干燥处
2. **避免潮湿**：湿度控制在40-60%
3. **定期通电**：每1-2个月开机一次
4. **屏幕保护**：使用软布包裹屏幕

**维修经验分享**：PSP 1000 的屏幕更换是所有 PSP 型号中最简单的，因为它的结构相对简单。最大的难点是屏幕排线，一定要小心操作。如果第一次拆机，建议先看拆机视频，了解每个步骤。
