# PSP 1000 电池更换完全指南

> 你的 PSP 1000 续航越来越短，充电后只能用1-2小时，或者干脆充不进电、自动关机。别担心，这是 PSP 1000 电池老化的典型症状。原装电池寿命约3-5年，更换电池能让你的 PSP 重获新生！

## 所需工具
- 十字螺丝刀，PH00规格
- 塑料撬棒或吉他拨片
- 新 PSP 1000 电池（型号：PSP-110）
- 无水酒精和棉签
- 防静电手套（可选）
- 工作垫

## 维修步骤

### 1. 安全准备与电池取出
1. **完全关机**：确保 PSP 完全关机，不是待机状态
2. **取出 UMD 光盘**：打开 UMD 仓，取出光盘
3. **打开电池仓盖**：滑动电池仓盖锁，打开盖子
4. **取出旧电池**：轻轻拉出电池，注意不要用力过猛
5. **断开连接**：电池通过金属触点连接，直接拔出即可

### 2. 新电池安装
1. **检查新电池**：确认电池型号为 PSP-110（PSP 1000 专用）
2. **清洁触点**：用棉签蘸无水酒精清洁电池仓内的金属触点
3. **插入新电池**：将新电池金属触点朝下，轻轻推入电池仓
4. **确认到位**：听到轻微"咔哒"声表示安装到位
5. **盖上仓盖**：滑动电池仓盖锁，确保盖子牢固

### 3. 首次充电与校准
1. **连接充电器**：使用原装 PSP 充电器（5V 2A）
2. **充电指示灯**：橙色灯亮起表示正在充电
3. **完全充电**：充电约3-4小时，直到橙色灯熄灭
4. **开机测试**：拔掉充电器，开机测试续航
5. **电池校准**：
   - 充满电后，正常使用直到自动关机
   - 再次充满电，重复2-3次完成校准

### 4. 常见问题解决
#### 问题1：新电池不充电
1. 检查充电器：确认充电器输出5V 2A
2. 检查充电接口：清洁 PSP 充电接口
3. 检查电池触点：确保电池触点清洁且接触良好
4. 尝试不同充电器：排除充电器故障

#### 问题2：电池续航短
1. **屏幕亮度**：降低屏幕亮度（默认最亮很耗电）
2. **无线功能**：关闭 WiFi（如果不使用）
3. **UMD 光驱**：减少 UMD 光盘使用（光驱耗电大）
4. **CPU 频率**：游戏时使用默认 333MHz，不要超频

#### 问题3：电池仓松动
1. 检查电池仓卡扣：是否有损坏
2. 检查电池尺寸：确保新电池尺寸正确
3. 临时解决：在电池背面贴一小片双面胶增加厚度

## PSP 1000 电池系统详解

### 🔌 充电电路特点
1. **智能充电**：PSP 有完善的充电管理芯片
2. **温度保护**：电池温度过高会自动停止充电
3. **过充保护**：充满后自动切换到涓流充电
4. **过放保护**：电量过低会自动关机保护电池

### 📊 续航优化技巧
1. **最佳亮度**：室内使用亮度调到2-3级即可
2. **省电模式**：在设置中开启自动亮度调节
3. **游戏选择**：2D游戏比3D游戏省电30-50%
4. **记忆棒速度**：使用高速记忆棒减少读取时间

### 📈 电池寿命管理
1. **循环次数**：锂电池约300-500次完整循环
2. **存放电量**：长期存放保持40-60%电量
3. **温度影响**：高温（>35°C）会加速电池老化
4. **更换周期**：建议每3-4年更换一次电池

### 🛠️ DIY 电池修复（仅限高手）
如果原装电池只是保护板故障：
1. 拆开电池外壳（小心操作）
2. 测量电芯电压，确认电芯是否完好
3. 更换保护板或重新激活
4. **风险警告**：锂电池有起火风险，非专业人士勿试
