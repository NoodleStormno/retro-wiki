# Super Nintendo外壳更换完全指南

![Super Nintendo 主机](images/supernintendo2.webp)

## 概述

Super Nintendo（SNES）的外壳保护着内部精密电子元件，但由于长期使用、不当存储或意外跌落，外壳可能出现裂纹、变色、磨损或损坏。本指南将详细介绍如何安全地更换SNES外壳，让您的主机焕然一新。

## 所需工具

- 十字螺丝刀（#00、#0、#1号）
- 平头螺丝刀或塑料撬棒
- 镊子
- 新的SNES外壳套装（包括上壳和下壳）
- 异丙醇和棉签
- 压缩空气罐
- 防静电腕带（推荐）
- 软布或毛巾（保护工作表面）
- 小容器（存放螺丝和小零件）

## 安全注意事项

1. **断电操作**：确保SNES主机完全断电，拔掉所有连接线
2. **静电防护**：处理内部电子元件时佩戴防静电腕带
3. **塑料保护**：SNES外壳塑料可能老化变脆，操作要格外小心
4. **部件保存**：拆卸时将所有螺丝、弹簧和小零件放在安全位置
5. **拍照记录**：拆卸前拍照，记录原始装配方式和部件位置

## 诊断与准备

### 步骤1：评估外壳状况
1. **检查损坏类型**：
   - **表面磨损**：划痕、变色、光泽度下降
   - **结构损坏**：裂纹、断裂、缺失部分
   - **功能影响**：卡扣损坏、螺丝孔滑丝、变形

2. **确定更换范围**：
   - 只需要更换上壳、下壳，还是全套更换
   - 检查按钮、端口开口是否匹配
   - 确认新外壳与您的SNES型号兼容

### 步骤2：选择新外壳
1. **材质选择**：
   - **ABS塑料**：原装材质，耐用性好
   - **再生塑料**：环保选择，质量可能参差不齐
   - **3D打印**：定制选择，需要精确建模

2. **颜色选择**：
   - **原装灰色**：保持经典外观
   - **透明外壳**：展示内部结构
   - **彩色外壳**：个性化选择
   - **特殊涂层**：哑光、亮光或其他表面处理

3. **质量检查**：
   - 检查注塑质量，有无毛边、缩水
   - 检查尺寸精度，与原件对比
   - 检查卡扣和螺丝孔是否完好

## 更换步骤

### 步骤1：完全拆卸主机

1. **准备工作**：
   - 拔掉所有外部连接线
   - 将主机放在柔软、干净的工作面上
   - 准备容器分类存放螺丝

2. **卸下底部螺丝**：
   - 使用#1十字螺丝刀卸下主机底部的所有螺丝
   - SNES通常有6-7颗底部螺丝
   - 注意螺丝长度可能不同，按位置分类存放

3. **分离外壳**：
   - 小心地抬起上壳
   - 注意上壳可能有内部卡扣
   - 如果有阻力，检查是否有未卸下的螺丝或卡扣

4. **断开内部连接**：
   - 断开电源开关连接线
   - 断开复位按钮连接线（如果有）
   - 注意连接器的方向和锁定机制

### 步骤2：拆卸内部组件

1. **取出主板**：
   - 卸下固定主板的螺丝
   - 小心抬起主板，注意不要弯曲
   - 断开所有与外壳连接的线缆

2. **拆卸电源板**（如果需要）：
   - 卸下电源板的固定螺丝
   - 断开电源输入连接器
   - 小心取出电源板

3. **拆卸卡带插槽**：
   - 如果卡带插槽与外壳有连接，小心拆卸
   - 注意弹簧和机械部件

4. **拆卸其他部件**：
   - 拆卸控制器端口（如果与外壳分离）
   - 拆卸RF调制器（早期型号）
   - 拆卸所有按钮和开关

### 步骤3：清洁和检查

1. **清洁旧部件**：
   - 用压缩空气清洁主板和所有电子部件
   - 用异丙醇清洁连接器触点
   - 检查所有部件是否完好，有无需要维修的

2. **检查新外壳**：
   - 检查新外壳的每个细节
   - 测试所有开口是否对齐
   - 检查卡扣和螺丝孔是否完好
   - 清洁新外壳内部，去除可能的生产残留

### 步骤4：安装到新外壳

1. **安装基础部件**：
   - 先将电源板安装到新下壳（如果适用）
   - 安装卡带插槽和相关机械部件
   - 安装控制器端口

2. **安装主板**：
   - 将主板小心放入新下壳
   - 对齐所有固定孔
   - 暂时不要拧紧所有螺丝

3. **安装按钮和开关**：
   - 安装电源开关
   - 安装复位按钮
   - 安装弹出按钮和相关弹簧
   - 测试每个按钮的动作是否顺畅

4. **连接内部线缆**：
   - 按照原始布局连接所有内部线缆
   - 确保连接器完全插入并锁定
   - 整理线缆，避免被夹住或干扰其他部件

### 步骤5：最终组装

1. **检查对齐**：
   - 检查所有部件是否正确就位
   - 检查所有开口是否对齐
   - 检查按钮和开关功能

2. **安装上壳**：
   - 将上壳对准下壳
   - 确保所有卡扣正确对接
   - 从一侧开始，逐渐合拢

3. **拧紧螺丝**：
   - 按照对角线顺序拧紧所有螺丝
   - 不要过度拧紧，避免损坏塑料螺纹
   - 使用适当大小的螺丝刀，避免滑丝

### 步骤6：功能测试

1. **外观检查**：
   - 检查外壳是否完全闭合，无缝隙
   - 检查所有按钮是否正常突出
   - 检查所有端口是否畅通

2. **基本功能测试**：
   - 连接电源和视频线
   - 打开电源开关，测试指示灯
   - 测试复位按钮
   - 测试弹出按钮

3. **全面功能测试**：
   - 插入游戏卡带，测试读取
   - 测试两个控制器端口
   - 测试视频和音频输出
   - 长时间运行测试稳定性

## 特殊型号注意事项

### SNS-001（初代SNES）
- 紫色电源和复位按钮
- 较圆润的外形设计
- 可能有不同的卡扣设计

### SNS-101（SNES Jr./精简版）
- 更小巧的外形
- 集成控制器端口
- 不同的内部布局

### 区域差异
- **北美版**：紫色按钮，凸起LOGO
- **日本版**：彩色按钮，平面LOGO
- **欧洲版**：不同电源要求，可能的外形调整

## 常见问题与解决方案

### 问题1：新外壳无法完全闭合
- **可能原因**：部件安装不当或外壳尺寸问题
- **解决方案**：检查每个部件是否正确就位，必要时轻微调整

### 问题2：按钮卡滞或不顺畅
- **可能原因**：按钮与外壳开口对齐问题
- **解决方案**：检查按钮安装，必要时轻微打磨开口边缘

### 问题3：螺丝孔不对齐
- **可能原因**：外壳制造公差或部件安装偏差
- **解决方案**：不要强行拧入，检查对齐，必要时使用较小螺丝或扩大孔位

### 问题4：内部线缆太短
- **可能原因**：新外壳内部布局略有不同
- **解决方案**：重新布线，或考虑延长线缆

## 定制与美化选项

### 喷漆定制
1. **表面准备**：彻底清洁，轻微打磨增加附着力
2. **底漆**：使用塑料专用底漆
3. **颜色漆**：多层薄涂，避免流挂
4. **保护漆**：使用透明保护漆增加耐用性

### 贴纸与装饰
1. **定制贴纸**：设计并打印个性化贴纸
2. **水转印**：复杂图案的理想选择
3. **UV打印**：直接在外壳上打印图案

### 功能增强
1. **LED改装**：添加状态指示灯或装饰灯光
2. **通风增强**：增加通风孔改善散热
3. **端口升级**：添加现代视频输出端口

## 维护建议

1. **日常清洁**：
   - 使用微纤维布清洁表面
   - 避免使用刺激性化学清洁剂
   - 定期清洁通风口

2. **存储建议**：
   - 存放在阴凉干燥处
   - 避免阳光直射，防止变色
   - 使用防尘罩保护

3. **使用习惯**：
   - 避免在主机上放置重物
   - 小心插拔连接线
   - 定期检查外壳完整性

## 技术规格

- **材料**：ABS塑料（原装）
- **颜色**：浅灰色（北美），多种颜色（其他地区）
- **尺寸**：约260mm × 200mm × 70mm（SNS-001）
- **重量**：约1.1kg（带内部部件）
- **表面处理**：纹理表面，减少指纹

## 专业技巧

1. **塑料处理**：
   - 如果塑料卡扣断裂，可使用塑料焊接修复
   - 使用塑料专用胶水进行粘接
   - 打磨时使用细砂纸，避免过度磨损

2. **螺丝管理**：
   - 使用磁化螺丝刀防止螺丝掉落
   - 制作螺丝位置图，确保正确安装
   - 如果螺丝孔滑丝，可使用塑料填料修复

3. **兼容性验证**：
   - 购买前确认外壳与您的SNES型号完全匹配
   - 检查按钮颜色和样式是否一致
   - 确认所有开口位置和尺寸正确

## 总结

更换SNES外壳是一项中等难度的维修工作，主要挑战在于小心处理内部部件和确保正确重新组装。通过本指南的详细步骤，您可以让老旧或损坏的SNES焕然一新。无论是修复损坏的外壳，还是进行个性化定制，耐心和细心都是成功的关键。如果您对拆卸电子设备不熟悉，建议先观看相关视频教程或寻求有经验的朋友帮助。
