# GBA屏幕烧屏换偏光膜教程

> **故障现象**：GBA屏幕出现永久性残影、颜色失真、对比度下降，在关机状态下也能看到淡淡的图像痕迹。

## 所需工具
- 三角撬棒
- 热风枪或电吹风
- 无尘布
- 偏光膜（45度角）
- 刮板
- 屏幕清洁液

## 维修步骤

### 1. 拆卸GBA外壳
1. 卸下背部6颗螺丝
2. 用三角撬棒小心分离前后壳
3. 断开屏幕排线连接
4. 取出屏幕总成

### 2. 移除旧偏光膜
1. 用热风枪低温加热屏幕边缘（80-100°C）
2. 用刮板轻轻撬起偏光膜一角
3. 缓慢均匀地撕下旧偏光膜
4. 用屏幕清洁液清理残留胶水

### 3. 粘贴新偏光膜
1. 在新偏光膜保护膜上标记45度角
2. 对齐屏幕边缘，先贴一边
3. 用刮板慢慢推平，排除气泡
4. 撕掉保护膜，检查贴合情况

### 4. 组装测试
1. 重新连接屏幕排线
2. 开机测试显示效果
3. 检查有无气泡或灰尘
4. 确认无误后组装外壳

## 注意事项
- 偏光膜角度必须准确（GBA为45度）
- 操作环境要无尘，避免灰尘进入
- 加热温度不要过高，避免损坏屏幕
- 如有气泡，可用针轻轻刺破后压平
