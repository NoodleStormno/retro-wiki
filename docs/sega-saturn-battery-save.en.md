# Sega Saturn Battery Replacement & Save Data Rescue

> Your Saturn has amnesia! Every power-on requires resetting time/date, game saves mysteriously disappear. 10 hours of *Sakura Wars* progress gone, *Shining Force III* class change records lost, worst of all *Sega Rally Championship* high score tables wiped clean. And that annoying "beep" battery warning sound on every startup makes you anxious.

## Required Tools
- Cross-head screwdriver, PH1 specification
- CR2032 coin cell battery, buy quality ones
- Battery holder, optional but highly recommended
- Multimeter, for checking voltage
- Tweezers, for removing battery
- Electrical tape, for insulation

## Repair Steps

### 1. Open Case and Assess Situation
1. Unplug all cables (Saturn has many, be careful)
2. Remove 6 screws from bottom (note different lengths)
3. Open top cover, locate motherboard
4. Find battery location - usually top right corner of motherboard

### 2. Old Battery Removal (Two Methods)
**Method A: Direct Replacement (Original Soldered)**
1. Use soldering iron to melt solder at both battery terminals
2. Remove old battery with tweezers
3. Clean solder pads, prepare for new battery

**Method B: Install Battery Holder (Permanent Solution)**
1. Remove old battery same as above
2. Solder battery holder (note polarity)
3. Future battery changes become plug-and-play, no soldering needed

### 3. New Battery Installation
1. **CR2032**, don't mistake for CR2025
2. Confirm polarity (marked on motherboard)
3. Solder or insert into battery holder
4. Measure voltage with multimeter, should be above 3.0V

### 4. System Settings Restoration
1. Power on, listen for "beep" warning sound
2. Enter system settings, adjust time and date
3. Set game region (Japanese/US)
4. Save settings, reboot and test

### 5. Save Data Backup Awareness
1. Purchase Saturn memory card (official or third-party)
2. Regularly backup important saves to memory card
3. Develop good habit: double backup important progress
