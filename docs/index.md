## 安全须知

维修前一定要读这几条：

1. 断电操作：所有维修前务必断开电源
2. 防静电：操作精密电路时佩戴防静电手套
3. 工具准备：使用合适的工具，避免损坏螺丝
4. 零件保管：拆卸时妥善保管所有螺丝和零件
5. 排线小心：拔插排线要轻柔，避免折断
