# PS1光头功率调节与读盘修复

> 老伙计的PS1开始挑食了！有些游戏盘读得顺，有些死活不认。开机后光盘转得飞快，但就是进不了游戏画面，有时候还会出现请插入PlayStation格式光盘的提示。

## 所需工具
- 十字螺丝刀，PH0规格
- 小号一字螺丝刀
- 无水酒精和棉签
- 万用表，可选，老鸟专用
- 防静电手环，保护主板用
- 强光手电筒

## 维修步骤

### 1. 开膛破肚，找到病灶
1. 拔掉所有线材，把PS1翻过来
2. 卸下底部的7颗螺丝（注意长短不一，最好记位置）
3. 小心打开外壳，别扯断排线
4. 找到光头组件——那个带着透镜的小车

### 2. 基础清洁，先礼后兵
1. 用吹气球吹去表面浮尘
2. 棉签蘸少量酒精，轻轻擦拭透镜
3. **重要**：一定要单向擦拭，别来回搓
4. 检查光头导轨，上点润滑脂

### 3. 调节功率，对症下药
1. 找到光头背面的电位器（通常标着VR）
2. 用一字螺丝刀**逆时针**微调15-30度
3. 装回外壳，接电测试
4. 如果还不行，再调15度，**千万别贪多**

### 4. 终极测试，验明正身
1. 用你最熟悉的游戏盘测试（我一般用《生化危机2》）
2. 测试不同区域的光盘（内圈外圈都试试）
3. 连续读盘30分钟，观察稳定性
4. 记录最佳调节位置，方便以后微调

## 老鸟经验谈

### 💡 调节心得
- **第一次调节**：先清洁，80%的问题清洁就能解决
- **调节幅度**：每次不超过30度，调多了直接烧头
- **测试方法**：先用CD音乐盘测试，再试游戏盘
- **终极方案**：调节无效就直接换光头，别折腾

### ⚠️ 避坑指南
1. **别用嘴吹光头**：口水蒸汽会腐蚀透镜
2. **别用普通酒精**：必须用99%无水酒精
3. **别在通电时调节**：小心触电烧机
4. **别买山寨光头**：寿命短，读盘差

### 🎮 游戏兼容性
- **早期PS1（1000型）**：调节效果明显，但光头易老化
- **后期PS1（7500型）**：光头耐用，但调节范围小
- **PSone迷你版**：光头难找，建议直接改硬盘
