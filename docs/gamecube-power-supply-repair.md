## 所需工具
- 三角螺丝刀（Y字头），GameCube专用
- 十字螺丝刀，PH0规格
- 万用表，数字式最好
- 电烙铁和焊锡
- 电容套装（电解电容为主）
- 热风枪（可选，用于拆焊）
- 无水酒精和棉签
- 防静电手套和工作垫

## 维修步骤

### 1. 安全拆机，取出电源板
1. 拔掉所有线材，记住AV线和电源线位置
2. 卸下底部6颗三角螺丝（Y字螺丝刀）
3. 打开上盖，小心光驱排线
4. 找到电源板——位于主机后部的大块电路板
5. 断开电源板与主板的连接线
6. 卸下固定电源板的螺丝，取出电源板

### 2. 电源板外观检查
1. **电容检查**：重点检查主滤波电容（大个的圆柱形电容）是否有鼓包、漏液
2. **保险丝检查**：找到板上的玻璃管保险丝，检查是否熔断
3. **整流桥检查**：检查4个二极管组成的整流桥
4. **开关管检查**：检查MOSFET开关管是否有烧焦痕迹
5. **电阻检查**：检查大功率电阻是否变色、开裂

### 3. 电压测量与诊断
1. **输入端测量**：测量电源接口处的电压，应为110V或220V（根据版本）
2. **保险丝测试**：用万用表通断档测试保险丝是否导通
3. **整流后电压**：测量整流桥输出端，应有约300V直流电压
4. **输出端测量**：测量电源板输出接口各引脚电压
   - 引脚1：+12V（光驱电机）
   - 引脚2：+5V（主板核心供电）
   - 引脚3：+3.3V（芯片供电）
   - 引脚4：GND（接地）
   - 引脚5：GND（接地）

### 4. 常见故障维修
#### 故障1：完全无反应
1. 检查保险丝，更换同规格保险丝（通常为2A）
2. 检查整流桥，如有损坏更换4个二极管
3. 检查开关管，更换同型号MOSFET

#### 故障2：开机几秒断电
1. 检查主滤波电容，更换鼓包电容
2. 检查反馈电路，特别是光耦和TL431
3. 检查输出端电容，更换老化电容

#### 故障3：电压不稳定
1. 检查稳压电路，特别是431基准电压源
2. 检查反馈电阻网络，测量阻值是否变化
3. 检查输出滤波电容，全部更换为低ESR电容

### 5. 电容更换指南
GameCube电源板常用电容规格：
- **主滤波电容**：400V 100μF（110V版）或 400V 47μF（220V版）
- **输出滤波电容**：
  - +12V输出：16V 470μF
  - +5V输出：10V 1000μF  
  - +3.3V输出：6.3V 470μF
- **建议品牌**：Rubycon、Nichicon、Panasonic

### 6. 组装与测试
1. 清洁电源板，去除助焊剂残留
2. 安装回主机，连接所有线缆
3. **先不盖盖子测试**：观察是否有异常发热、冒烟
4. 测试开机：指示灯应亮起，光驱应有动作
5. 放入光盘测试读盘功能
6. 连续运行30分钟，测试稳定性

- **日版/美版**：110V输入，主滤波电容100μF
- **欧版/澳版**：220V输入，主滤波电容47μF
- **国行版**：220V输入，比较罕见

- **DOL-101**：阉割版，无数字口
- **电源板通用性**：大部分型号可以互换

## 安全注意事项

### ⚠️ 高压警告！
1. **断电操作**：维修前必须拔掉电源线
2. **电容放电**：大电容存储高压，维修前必须放电
3. **隔离工作**：使用绝缘工作垫，避免触电
4. **一人操作**：维修时不要让他人触碰

### 🔧 焊接技巧
1. **温度控制**：电烙铁350度左右
2. **时间控制**：每个焊点不超过3秒
3. **清理焊盘**：使用吸锡器或吸锡线
4. **检查虚焊**：焊接后检查每个焊点
