## 详细维修步骤

我在秋叶原修了20年GameBoy，总结出这套方法。跟着做，别着急。

### 第一步：安全拆机

拆机最怕的就是损坏其他部件。慢慢来：

1. 先卸下电池盖，取出里面的2节AAA电池
2. 拆除6颗螺丝，4颗Y字型螺丝在背面，还有2颗十字螺丝藏在电池仓的贴纸下面
3. 用塑料撬棒从充电口那里轻轻撬开外壳
4. 断开主板和液晶屏连接的排线
5. 卸下固定主板的2颗螺丝，取出主板

### 第二步：按键系统诊断

取出主板后，检查按键系统：

1. **方向键检查**：
   - 上、下、左、右四个方向
   - 检查橡胶垫是否老化开裂
   - 检查触点是否氧化磨损
   - 检查按键帽是否磨损

2. **AB键检查**：
   - A键和B键
   - 检查橡胶垫的凸点是否磨损
   - 检查弹簧（如果有）是否失效
   - 检查按键帽是否松动

3. **Select/Start键检查**：
   - 这两个键是微动开关
   - 检查开关是否损坏
   - 检查焊点是否开裂

4. **电路板检查**：
   - 检查按键对应的触点是否氧化
   - 检查走线是否断裂

### 第三步：橡胶垫更换

橡胶垫老化是最常见的问题：

1. **拆除旧橡胶垫**：
   - 方向键是一个整体的十字橡胶垫
   - AB键是两个独立的橡胶垫
   - 用镊子轻轻取下

2. **清洁触点**：
   - 用棉签蘸异丙醇清洁电路板上的触点
   - 每个触点都要清洁到位

3. **安装新橡胶垫**：
   - 方向键橡胶垫要对准十字轴
   - AB键橡胶垫要对准位置
   - 确保橡胶垫完全贴合

4. **测试**：安装后先测试按键反应

### 第四步：触点修复

如果橡胶垫没问题，可能是触点问题：

1. **检查触点磨损**：
   - 在放大镜下检查触点是否磨损
   - 磨损的触点会失去导电性

2. **清洁触点**：
   - 用异丙醇彻底清洁
   - 用橡皮擦轻轻擦拭（非常小心）

3. **导电银浆修复**：
   - 如果触点磨损严重
   - 用牙签蘸少量导电银浆
   - 在磨损处涂上薄薄一层
   - 室温固化24小时

4. **测试**：固化后测试导电性

### 第五步：微动开关维修

Select/Start键是微动开关：

1. **检查开关**：
   - 按下时应该有清脆的咔嗒声
   - 用万用表测量通断

2. **清洁开关**：
   - 喷入少量电子清洁剂
   - 反复按压几次

3. **更换开关**：
   - 如果开关损坏
   - 用烙铁拆除旧开关
   - 焊接新开关（注意方向）

### 第六步：按键帽维修

按键帽磨损或松动：

1. **检查按键帽**：
   - 方向键的十字帽是否磨损
   - AB键的圆形帽是否松动

2. **清洁按键帽**：
   - 用异丙醇清洁
   - 检查卡扣是否完好

3. **更换按键帽**：
   - 如果磨损严重
   - 购买原装或副厂替换
   - 对准位置按压到位

### 第七步：组装测试

1. 维修完成后，先不要装机测试
2. 连接电池，测试所有按键：
   - 方向键四个方向
   - AB键按压和回弹
   - Select/Start键反应
3. 测试通过后，按拆机逆序装回
4. 运行游戏测试：
   - 俄罗斯方块测试方向键
   - 口袋妖怪测试AB键和菜单键
5. 最后做压力测试，连续按压100次

## 老师傅的叮嘱

在秋叶原开店20年，客人常问这些问题，我一个个回答你。

### 为什么Game Boy Pocket按键容易失灵？

三个主要原因。第一是橡胶老化，20多年的橡胶早就硬化失去弹性。第二是触点氧化，空气中的水分导致氧化。第三是使用频率，动作游戏对按键磨损很大。

### 橡胶垫多久需要更换？

看使用频率。轻度使用可能10年才需要换。重度使用可能3-5年就要换。如果出现按键不灵、回弹慢、需要用力按，就是该换了。

### 导电银浆修复能坚持多久？

如果做得好，能坚持2-5年。关键是清洁要彻底，银浆要涂得薄而均匀。如果涂得太厚，容易脱落。如果清洁不彻底，导电性不好。

### 如何保养按键？

我建议定期清洁，用棉签蘸异丙醇清洁按键缝隙。避免在灰尘大的环境使用。不要用指甲用力按压，用指腹按压。长时间不用要放在防尘盒里。

## 维修难度

技术难度三颗星，需要细心操作。工具要求三颗星，需要一些专业工具。时间成本大概1-3小时，看损坏程度。成功率很高，90%以上。

## 我的终极建议

在秋叶原修了20年，我总结出几条经验：

1. 先诊断后动手，确定是橡胶问题还是触点问题
2. 从简单到复杂，先换橡胶垫再考虑其他
3. 清洁是关键，触点不清洁什么都白搭
4. 测试要全面，每个按键每个方向都要测试
5. 耐心最重要，按键系统很精密
