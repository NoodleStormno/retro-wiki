# PSP 1000 Screen Replacement Complete Guide

> Your PSP 1000 screen has dead pixels, bright lines, uneven backlight, or simply won't display. As the first-generation PSP model, its 4.3-inch TFT screen is a classic, but after years of use, issues are inevitable. Don't worry‚Äîscreen replacement is one of the most common PSP repair jobs!

## Required Tools
- Phillips screwdriver, PH00 size
- Plastic spudger or guitar pick
- New PSP 1000 screen assembly
- Isopropyl alcohol and cotton swabs
- Precision tweezers
- Anti-static gloves and work mat
- Hair dryer or heat gun (optional, for adhesive removal)

## Repair Steps

### 1. Safe Disassembly Preparation
1. **Complete power down**: Remove battery and UMD disc
2. **Remove all screws**:
   - 6 screws on back (4 in battery compartment, 2 in UMD compartment)
   - Note different screw lengths, mark them
3. **Separate housing**: Use plastic spudger to gently pry open from edges
4. **Be careful with flex cables**: Watch for screen and button flex cables when opening

### 2. Screen Assembly Removal
1. **Locate screen flex cable**: On right side of motherboard, white or brown cable
2. **Open flex cable latch**: Gently lift latch with tweezers
3. **Remove screen flex cable**: Pull out gently, don't use excessive force
4. **Remove screen mounting brackets**:
   - Metal frame secures screen around edges
   - Remove mounting screws (usually 4)
5. **Remove old screen**: Carefully take out entire screen assembly

### 3. New Screen Installation
1. **Inspect new screen**: Confirm it's specifically for PSP 1000
2. **Clean mounting surface**: Clean screen area with isopropyl alcohol
3. **Install screen**: Place new screen into frame, align screw holes
4. **Secure screen**: Install mounting screws with even pressure
5. **Connect flex cable**: Insert screen flex cable, press latch to lock

### 4. Screen Testing
1. **Initial test**: Test without reassembling case, connect battery and power on
2. **Check display**: Look for any display abnormalities
3. **Test touch**: PSP 1000 has no touchscreen, skip this step
4. **Check backlight**: Ensure backlight is even, no dark corners
5. **Test colors**: Display solid color images to check for dead pixels and color shift

### 5. Reassembly & Final Testing
1. **Clean interior**: Remove dust with cotton swabs
2. **Reassemble**: Follow disassembly steps in reverse order
3. **Return screws**: Pay attention to screw lengths and positions
4. **Final test**: Test all functions after complete reassembly

## Common Screen Issues & Solutions

### Issue 1: Dead Pixels
1. **Dead pixel types**:
   - Bright pixel: Always lit
   - Dark pixel: Never lights up
   - Stuck pixel: Shows fixed color
2. **Repair attempt**: Use dead pixel fix software (low success rate)
3. **Final solution**: Replace screen

### Issue 2: Bright Lines
1. **Horizontal lines**: Usually flex cable issue
2. **Vertical lines**: Likely screen damage
3. **Multiple lines**: Screen driver chip failure
4. **Repair solution**: Replace screen assembly

### Issue 3: Uneven Backlight
1. **Edge darkening**: Backlight aging or installation issue
2. **Local dark spots**: Backlight guide plate problem
3. **Overall dimness**: Backlight end of life
4. **Solution**: Replace screen or backlight only

### Issue 4: Screen Flickering
1. **Regular flickering**: Backlight driver circuit issue
2. **Random flickering**: Flex cable contact problem
3. **Color flickering**: Screen driver chip failure
4. **Troubleshooting steps**: Check flex cable ‚Üí Check motherboard ‚Üí Replace screen

### üõ†Ô∏è Screen Flex Cable Handling Tips
Screen flex cables are very fragile‚Äîhandle with extreme care:

#### Flex Cable Insertion/Removal
1. **Fully open latch**: Ensure latch is completely lifted
2. **Parallel removal**: Pull out parallel, don't twist
3. **Check contacts**: Inspect gold contacts for cleanliness and integrity
4. **Reinsert**: Align connector, gently push all the way in

#### Flex Cable Protection
1. **Avoid folding**: Don't bend flex cable excessively
2. **Anti-static**: Touch metal object to discharge static before handling
3. **Clean contacts**: Clean connector with isopropyl alcohol
4. **Secure position**: Ensure proper cable routing during reassembly

### üîß Screen Frame & Adhesive Handling
PSP 1000 screens use double-sided adhesive:

#### Old Adhesive Removal
1. **Heat to soften**: Use hair dryer on low heat to warm adhesive
2. **Slow separation**: Use plastic spudger to slowly separate
3. **Clean residue**: Remove leftover adhesive with isopropyl alcohol
4. **Avoid scratches**: Don't use metal tools

#### New Adhesive Installation
1. **Choose adhesive**: Use 3M double-sided tape or dedicated screen adhesive
2. **Cut to size**: Cut adhesive strips to fit screen borders
3. **Apply evenly**: Ensure adhesive is applied flat without bubbles
4. **Press to secure**: Press screen firmly after installation
