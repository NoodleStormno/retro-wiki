# Nintendo 64卡带电池更换完全指南（不丢失存档）

> 你的《塞尔达传说：时之笛》存档又没了？《黄金眼007》的解锁内容全部消失？N64卡带的CR2032电池寿命约5-10年，时间一到，所有存档都会"蒸发"。但别绝望！有一种方法可以更换电池而不丢失珍贵存档。掌握这门技术，你的游戏记忆将永存！

## 所需工具
- 3.8毫米游戏比特螺丝刀（必须！）
- 电烙铁，25-40W，尖头
- 焊锡丝和助焊剂
- 吸锡器或吸锡线
- 镊子，精细尖头
- 外部电源：3V电池盒或可调电源
- 导线和夹子（鳄鱼夹）
- 万用表，检测电压
- 防静电垫和工作台
- 新CR2032电池（松下或索尼品牌）

## 维修步骤

### 1. 准备工作（最关键！）
1. **确认电池没电**：用万用表测电池电压（低于2.5V需要更换）
2. **准备外部电源**：3V直流电源，正负极正确
3. **测试外部电源**：确保稳定输出3V
4. **准备所有工具**：一次性拿齐，避免中途离开
5. **清洁工作区**：避免灰尘污染
6. **心理准备**：操作需要耐心和稳定手法

### 2. 安全拆卸卡带
1. 使用3.8毫米游戏比特螺丝刀卸下底部螺丝
2. 用塑料撬棒轻轻分离外壳（注意4个卡扣）
3. 取出电路板，放在防静电垫上
4. **不要断开任何连接**：保持电路板完整
5. 定位电池：通常在电路板中央或边缘

### 3. 连接外部电源（保住存档的关键！）
1. 用万用表确认电池正负极
2. 将外部电源正极连接到电池正极焊点
3. 将外部电源负极连接到电池负极焊点
4. **重要**：先夹负极，再夹正极
5. 确认连接牢固，不会意外脱落
6. 用万用表检测：电路板应有3V电压

### 4. 焊接更换电池
1. 用电烙铁加热电池正极焊点
2. 用镊子轻轻取下旧电池（外部电源保持连接）
3. 清理正极焊盘，去除旧焊锡
4. 同样方法取下负极，清理焊盘
5. **保持外部电源连接**：此时外部电源为SRAM供电

### 5. 安装新电池
1. 准备新CR2032电池，确认极性
2. 将新电池放在正确位置
3. 先焊接负极：快速准确，避免过热
4. 再焊接正极：同样快速完成
5. **焊接时保持外部电源连接**
6. 检查焊点：应光滑饱满，无虚焊

### 6. 断开外部电源与测试
1. 先断开外部电源正极
2. 再断开外部电源负极
3. 用万用表测试新电池电压：应有3V左右
4. 测试存档：暂时组装卡带，插入主机
5. 确认所有存档完好无损
6. 完全组装卡带，拧紧螺丝
