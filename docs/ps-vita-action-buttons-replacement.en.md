# PlayStation Vita Action Buttons Replacement Complete Guide

## Overview

The PlayStation Vita action buttons (△○×□) are core components for game control. After prolonged use, they may develop issues like unresponsiveness, weak rebound, or sticking. This guide provides detailed instructions for replacing PS Vita action buttons.

## Required Tools

- Cross-head screwdriver (PH00)
- Plastic spudger
- Tweezers
- New PS Vita action button set (△○×□)
- Anti-static wrist strap (recommended)

## Safety Precautions

1. **Power off completely**: Ensure PS Vita is fully powered down
2. **Static protection**: Discharge static electricity before operation
3. **Ribbon cable care**: Button circuit board ribbon cables are fragile
4. **Silicone pad protection**: Handle rebound-providing silicone pads carefully

## Detailed Steps

### Step 1: Remove Back Panel

1. Remove back panel according to your model
2. Carefully detach back panel

### Step 2: Complete Power Disconnection

1. Locate battery connector
2. Disconnect battery
3. Ensure device is completely powered off

### Step 3: Access Action Buttons

1. Action buttons located on right side of device
2. May require partial frame disassembly
3. Expose action button area

### Step 4: Remove Old Buttons

1. Locate button circuit board ribbon cable connector
2. Use tweezers to gently lift ribbon cable latch
3. Remove button ribbon cable
4. Remove screws securing button frame
5. Remove old button frame and buttons

### Step 5: Install New Buttons

1. Install new silicone pads
2. Place new △○×□ buttons into corresponding positions
3. Note button orientation: △ top, ○ right, × bottom, □ left
4. Install button frame
5. Tighten securing screws
6. Connect button ribbon cable

### Step 6: Testing and Reassembly

1. Temporarily connect battery to test each button function
2. Test button tactile feel and rebound
3. Reassemble all components
4. Install back panel

## Function Testing

1. **Individual button test**: Test each △, ○, ×, □ button
2. **Combination button test**: Test common game combinations
3. **Rapid pressing test**: Test response speed
4. **Game test**: Test in actual gameplay
