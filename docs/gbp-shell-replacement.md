## 准备工具

秋叶原维修店的标准配置是这样的：

- **精密螺丝刀套装**：日本制的YATO YT-6991，需要Y字型螺丝刀（Y00规格）
- **塑料撬棒套装**：不同尺寸的撬棒，避免划伤外壳
- **镊子**：弯头和平头各一把
- **异丙醇**：99%纯度，清洁外壳和内部
- **棉签**：清洁细小部位
- **防静电布**：擦拭屏幕和内部
- **热风枪或电吹风**：软化双面胶（可选）
- **新的外壳套装**：前后壳都要准备
- **螺丝收纳盒**：分类存放拆下的螺丝
- **工作垫**：防静电垫，保护外壳

## 详细维修步骤

我在秋叶原修了20年GameBoy，总结出这套方法。跟着做，别着急。

### 第一步：安全拆机

拆机最怕的就是划伤外壳或损坏内部元件。慢慢来：

1. **准备工作台**：铺上防静电垫，准备好所有工具
2. **取出电池**：先卸下电池盖，取出里面的2节AAA电池
3. **拆除螺丝**：
   - 4颗Y字型螺丝在背面四角
   - 2颗十字螺丝藏在电池仓的贴纸下面（小心撕开贴纸）
4. **分离外壳**：
   - 用塑料撬棒从充电口那里轻轻撬开
   - 沿着边缘慢慢移动，听到卡扣松开的声音就停一下
   - 不要用金属工具，会留下划痕
5. **断开连接**：
   - 断开主板和液晶屏连接的排线
   - 用指甲或者塑料片轻轻撬起排线锁扣

### 第二步：内部元件转移

这是最关键的一步，要小心操作：

1. **取出主板**：卸下固定主板的2颗螺丝，小心取出主板
2. **取出屏幕组件**：包括液晶屏、反光板、导光板（如果有）
3. **取出按键板**：方向键、AB键、Select/Start键
4. **取出电源开关和调节轮**：音量轮和对比度轮
5. **取出扬声器**：小心断开连接
6. **分类存放**：所有元件按顺序摆放，拍照记录原始位置

### 第三步：清洁与检查

在装到新外壳前，先清洁所有元件：

1. **主板清洁**：用异丙醇和棉签清洁主板，特别是电池触点和连接器
2. **屏幕清洁**：用防静电布轻轻擦拭液晶屏，不要用酒精直接喷
3. **按键清洁**：清洁按键触点，检查橡胶垫是否老化
4. **外壳内部**：检查新外壳内部是否有毛刺或瑕疵
5. **螺丝孔检查**：确保新外壳的螺丝孔位正确

### 第四步：安装到新外壳

按逆序安装，但要更小心：

1. **安装扬声器**：先装扬声器，注意方向
2. **安装电源开关和调节轮**：
   - 音量轮和对比度轮要对准孔位
   - 电源开关要安装到位
3. **安装按键板**：
   - 方向键的十字轴要对准
   - AB键的橡胶垫要放平
   - Select/Start键的弹簧（如果有）不要丢失
4. **安装屏幕组件**：
   - 先放反光板，再放导光板（如果有）
   - 最后放液晶屏，排线要预留足够长度
5. **安装主板**：
   - 对准所有螺丝孔位
   - 连接屏幕排线，锁紧锁扣
   - 连接扬声器线
6. **测试**：先不要上螺丝，连接电池测试所有功能

### 第五步：合盖与最终测试

测试通过后，正式合盖：

1. **对齐外壳**：前后壳要对齐所有卡扣位置
2. **轻轻按压**：从一侧开始，听到卡扣到位的声音
3. **安装螺丝**：
   - 先装背面的4颗Y字螺丝，不要拧太紧
   - 再装电池仓的2颗十字螺丝
4. **最终测试**：
   - 开机测试
   - 测试所有按键
   - 测试声音
   - 测试电源开关
   - 运行游戏测试
5. **贴回贴纸**：如果原来的贴纸还能用，小心贴回

## 老师傅的叮嘱

在秋叶原开店20年，客人常问这些问题，我一个个回答你。

### 为什么Game Boy Pocket外壳容易损坏？

三个主要原因。第一是材质，ABS塑料经过20多年会变脆。第二是使用频率，掌机经常携带，容易磕碰。第三是保存环境，阳光直射会导致变色，潮湿会导致霉斑。

### 原装外壳和副厂外壳有什么区别？

区别很大。原装外壳：
- 材质更好，更耐用
- 颜色准确，特别是限量版
- 卡扣设计精准，合盖严密
- 有任天堂的质感

副厂外壳：
- 价格便宜
- 颜色选择多
- 但材质可能较薄
- 卡扣可能不精准
- 颜色可能有色差

### 更换外壳会影响机器价值吗？

看情况。如果是普通版换原装外壳，影响不大。如果是限量版换副厂外壳，价值会大打折扣。如果是维修性更换（比如外壳开裂必须换），并且使用原装或高质量副厂外壳，对价值影响较小。

### 如何保养外壳？

我建议使用保护套，减少直接磕碰。存放时避免阳光直射。定期清洁，用软布擦拭。不要使用酒精等有机溶剂清洁，会导致塑料变白。

技术难度三颗星，主要是细心和耐心。工具要求两颗星，基本工具就行。时间成本大概1-2小时，第一次尝试可能更久。成功率很高，95%以上。

## 我的终极建议

在秋叶原修了20年，我总结出几条经验：

1. 拍照记录，拆一步拍一张，装的时候对照
2. 螺丝分类，不同位置的螺丝分开放
3. 温柔操作，塑料卡扣很脆弱
4. 测试先行，装到一半就测试，避免全部装完才发现问题
5. 接受不完美，20多年的老机器，新外壳也可能有小瑕疵

最后说几句心里话。外壳是Game Boy Pocket的脸面。很多客人拿来换外壳，不是为了卖钱，是为了找回当年的感觉。那台陪他度过童年的灰色Pocket，换上新外壳，就像回到了1996年。

记住，你修复的不仅是一台游戏机，是一段青春。慢慢来，你能行。
