# GameBoy Pocket 液晶横线修复完全指南

秋叶原20年老师傅亲授

> 你的GameBoy Pocket开机后屏幕出现一条或多条顽固的横线，有时固定不动，有时闪烁不停。玩口袋妖怪时皮卡丘被横线切成两半，俄罗斯方块的方块在横线处消失。最气人的是，这些横线不会随时间消失，反而可能越来越多。

## 准备工具

秋叶原维修店的标准配置是这样的：

- **精密螺丝刀套装**：推荐日本制的YATO YT-6991，或者同等品质的
  - 特别注意需要Y字型螺丝刀，Y00规格才能拆解GameBoy Pocket
- **电烙铁**：白光FX-888D，温度设定在320度最合适
- **焊锡**：千住无铅焊锡，0.6毫米粗细正好
- **吸锡器或者吸锡线**，两者都备着
- **万用表**：用来测量液晶排线连接
- **放大镜台灯**：5倍放大，能看清排线细节
- **异丙醇**：99%纯度，配上棉签
- **导电银浆**：备用方案要用到
- **热风枪**：可选但推荐，用来软化旧胶
- **防静电手环**：保护娇贵的液晶屏

## 详细维修步骤

我在秋叶原修了20年GameBoy，总结出这套方法。跟着做，别着急。

### 第一步：安全拆机

拆机最怕的就是造成二次伤害。慢慢来：

1. 先卸下电池盖，取出里面的2节AAA电池
2. 拆除6颗螺丝，4颗Y字型螺丝在背面，还有2颗十字螺丝藏在电池仓的贴纸下面
3. 用塑料撬棒从充电口那里轻轻撬开外壳，别用金属工具
4. 断开主板和液晶屏连接的排线，用指甲或者塑料片轻轻撬起那个锁扣

### 第二步：液晶屏拆解与诊断

打开后就能看到内部结构了：

1. 卸下固定主板的2颗螺丝，把主板取出来
2. GameBoy Pocket用的是反射式液晶，小心取下反光板
3. 仔细检查排线连接，主排线连接主板和液晶驱动芯片，侧边排线控制水平扫描线
4. 判断横线类型：
   - 如果是单条固定横线，通常是某条水平扫描线断了
   - 多条闪烁的横线，可能是排线接触不良或者驱动芯片有问题
   - 半屏都是横线，那很可能是驱动芯片的某组输出故障了

### 第三步：核心维修，三种方法选一种

#### 方法一：热压修复

成功率大概七成，适合新手：

1. 用异丙醇清洁液晶屏和排线连接处
2. 热风枪设定120度，低风速
3. 在排线连接处垫上耐热胶带，用热风枪均匀加热10到15秒
4. 用平整的工具比如信用卡轻轻按压
5. 冷却后装机测试，看看横线消失了没有

#### 方法二：导电银浆修复

成功率能到八成五，我比较推荐：

1. 用万用表蜂鸣档找到断开的线路
2. 棉签蘸异丙醇彻底清洁断点
3. 牙签蘸一点点导电银浆，在断点处画一条细线连接两端，宽度别超过0.5毫米
4. 室温下放24小时让它完全固化

#### 方法三：飞线修复

成功率最高，九成五，但难度也最大：

1. 用万用表确认断线的两端
2. 准备0.1毫米直径的漆包线，长度剪合适
3. 烙铁温度降到280度，在断线两端点上焊锡
4. 焊接漆包线，一定要牢固
5. 用UV胶或者绿油做绝缘处理

### 第四步：驱动芯片检查

如果排线没问题，那可能是芯片：

1. GameBoy Pocket用的是夏普LM24014T液晶驱动芯片
2. 检查电压：Vcc要在3.3伏左右，Vee是负15伏很关键，Vdd是5伏
3. 真要换芯片的话，用热风枪350度拆除旧的，清理焊盘涂上焊膏，对位放新芯片再用热风枪回焊

### 第五步：组装测试

1. 按拆机的逆序一步步装回去
2. 开机测试，看横线是不是完全消失了，测试所有按键功能，运行游戏检查显示完整性
3. 最后做个老化测试，连续运行1小时确认稳定性

## 老师傅的叮嘱

在秋叶原开店20年，客人常问这些问题，我一个个回答你。

### 为什么GameBoy Pocket特别容易出横线？

三个主要原因。第一是排线设计问题，Pocket版为了做轻薄，排线比原版GameBoy细多了。第二是老化，20多年过去了，排线胶水早就失效了。第三是使用习惯，频繁开合上盖导致排线反复弯折，不断就怪了。

### 修复后能坚持多久？

看你怎么修。热压修复大概能撑6个月到2年，有可能复发。导电银浆修复比较稳定，2到5年没问题。飞线修复最持久，5年以上接近永久。换芯片的话能用10年以上，但成本也最高。

### 有没有预防横线的方法？

当然有。我建议别频繁拆装，非必要不拆机。存放环境很重要，放防潮箱里，湿度控制在40到50。使用时要轻拿轻放，别挤压。每半年开机测试一次，有问题早发现。

### 值不值得修？还是直接换屏？

这得算经济账。修复成本大概500到1500日元，人工加材料。换屏要2500到4000日元，屏加人工。但原装屏永远比副厂屏值钱，有收藏价值。要是你的第一台GameBoy，那份情感价值更不是钱能衡量的。

## 维修难度

技术难度四颗星，需要很精细的操作。工具要求也是四颗星，专业工具少不了。时间成本大概2到4小时，第一次尝试可能更久。成功率看方法，70%到95%都有可能。

## 我的终极建议

在秋叶原修了20年，我总结出几条经验：

1. 先诊断后动手，用万用表彻底检查再决定怎么修
2. 备件提前准备好，导电银浆和漆包线都买齐
3. 耐心最重要，GameBoy Pocket的排线比头发还细，急不得
4. 测试是关键，每步完成后立即测试，避免错误累积
5. 接受不完美，20多年的老机器，修复后可能有轻微痕迹，那是岁月的证明

最后说几句心里话。在秋叶原修了20年GameBoy，每一台横线的Pocket背后都有一个故事。可能是孩子省下零花钱买的生日礼物，可能是父亲传给儿子的童年回忆。我们修复的不仅是屏幕，是连接两代人的游戏情怀。

记住，横线可以修，回忆不能丢。慢慢来，你能行。
